Play the classic **Snake game** directly in your terminal. Written in **Python**, this project demonstrates clean, modular code, making it easy to run, understand, and extend. Perfect for beginners and coding portfolios.  

---

## Features

- Terminal-based gameplay (Linux, macOS, Windows)  
- Smooth snake movement with `W`, `A`, `S`, `D` keys  
- Eat food to grow your snake  
- Collision detection with walls and self  
- Modular, beginner-friendly Python code  
- Easy to extend with additional features
